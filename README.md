# Трекер

Личный фитнес-трекер с DeepSeek-интеграцией. Один HTML-файл, vanilla JS, без сборки, без backend.

## Что внутри

- **Тренировочный плеер** — 4 программы (A/B/C/D), пошаговые таймеры с прогресс-кольцом, боксёрские раунды со случайными комбинациями, отсчёт 3-2-1 перед интервалами.
- **Прогрессия по неделям** — state machine: техника → темп → отягощение → совмещение. Уровни разблокируются по фактическим данным (болевые флаги), а не по календарю.
- **Тест восстановления бокса** — после 2 проходов `boxing_tier` растёт, отдых между раундами сокращается на 15 сек/cap на tier=5.
- **Дневник + DeepSeek-парсер** — свободный текст → структурированный JSON (калории, белок, настроение, RPE). При `confidence=low` или ошибке API — ручная форма.
- **Ежедневный совет** — DeepSeek комментирует фактические vs целевые показатели (кэш на день).
- **Прогресс** — график веса с 7-дн. скользящим средним, замеры талии/бёдер, стрик-календарь за 28 дней.
- **PWA** — устанавливается на главный экран, работает offline (service worker).

## Локальный запуск

Откройте `index.html` в браузере (двойной клик). DeepSeek-функции потребуют API-ключ (указывается в настройках или в онбординге при первом старте).

## Деплой на Render

Файл `render.yaml` уже настроен для Render Static Site.

1. Залейте этот репозиторий на GitHub.
2. На [render.com](https://render.com): **New +** → **Static Site** → подключите GitHub-репо.
3. Выберите ветку `main` — Render сам подхватит `render.yaml`:
   - Build Command: *(пусто)*
   - Publish Directory: `.`
4. Нажмите **Create Static Site**. URL вида `https://tracker-xxxx.onrender.com` будет готов через ~30 секунд.

Альтернатива без `render.yaml`: вручную укажите Build Command пустым, Publish Directory = `.`.

## Данные

Все данные хранятся в `localStorage` браузера. В настройках есть экспорт/импорт JSON-бэкапа. Скрытый dev-режим (5 тапов по версии в настройках или `?dev=1` в URL) открывает сброс `program_state`.

## Стек

- Один `index.html` со встроенными CSS/JS.
- `manifest.webmanifest`, `sw.js`, `icon-192.png`, `icon-512.png`, `favicon.svg` — для PWA.
- `render.yaml` — конфиг деплоя.
- Никаких npm-зависимостей, никакого bundler.
